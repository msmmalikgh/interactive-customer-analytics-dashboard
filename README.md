# interactive-customer-analytics-dashboard
A Streamlit-based interactive dashboard for Customer Analytics. Upload raw retail data to automatically compute RFM segmentation, estimate CLTV, visualize key metrics, and generate downloadable segment-wise reports.

This project provides a user-friendly Streamlit dashboard for retail customer analytics. Features include:

- RFM (Recency, Frequency, Monetary) calculation and scoring
- Customer segmentation into Champions, Loyal, Potential, At Risk, and Hibernating
- Customer Lifetime Value (CLTV) estimation
- Interactive visualizations: heatmaps, CLTV distribution, and scatterplots
- Key metrics summary (total customers, average recency, frequency, CLTV)
- Download filtered data per segment as Excel
- Fully self-updating dashboard upon Excel upload
